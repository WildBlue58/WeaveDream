## PropertyEditor

属性编辑器是织梦（WeaveDream）的核心组件之一，负责编辑选中组件的属性。

### 特性

- 支持多种组件类型的属性编辑
- 类型安全的属性配置
- 可扩展的编辑器架构
- 实时属性更新
