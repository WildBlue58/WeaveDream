# 项目文档

本目录包含项目的所有文档，按类别组织。

## 🎯 项目亮点

织梦（WeaveDream）是一个企业级可视化低代码编辑器，具有以下特点：

- ✨ **完善的架构设计**：模块化、组件化、插件化的分层架构
- 🚀 **优秀的代码质量**：严格的代码规范、文件行数限制、单一职责原则
- 🔧 **高度的可扩展性**：基于注册表模式的插件化设计，易于扩展
- 💎 **完整的类型安全**：全面的 TypeScript 类型系统
- 📚 **规范的文档体系**：详细的架构文档、开发指南和使用说明
