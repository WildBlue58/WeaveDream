## 快速开始

### 开发环境

1. 安装依赖：

```bash
pnpm install
```

2. 启动开发服务器：

```bash
pnpm dev
```

### 项目结构

详细的项目结构说明请参考 [架构文档](../architecture/PROJECT_STRUCTURE.md)。

## 工作流程

### 添加新功能

1. 创建 OpenSpec 变更提案（如需要）
2. 实现功能代码
3. 编写或更新相关文档
4. 测试和代码审查

### 提交代码

- 提交信息使用简体中文
- 遵循提交信息格式规范
- 确保通过 lint 检查
