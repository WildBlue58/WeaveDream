# 变更记录

本目录用于存放项目变更记录的文档。

> 📋 实际的变更提案和任务记录位于 `../openspec/changes/` 目录。

## 变更提案

所有变更提案都使用 OpenSpec 规范进行管理，包括：

- **提案文档** (`proposal.md`) - 变更的概述和目标
- **任务列表** (`tasks.md`) - 实现任务清单
- **规范文档** (`specs/`) - 详细的技术规范

## 查看变更

访问 [OpenSpec 变更目录](../../openspec/changes/) 查看所有变更提案的详细信息。

## 变更类别

- **功能增强** - 新增功能或扩展现有功能
- **UI/UX 优化** - 界面和用户体验改进
- **代码重构** - 代码结构优化和重构
- **Bug 修复** - 问题修复和改进
